How To SetUp

Tutorial:

  1. 1.Download the zip file to your computer for the Chrome extension you want to install.
  2. 2.Go to chrome://extensions/ and check the box for Developer mode in the top right.
  3. 3.Locate the ZIP file on your computer and unzip it.
  4. 4.Go back to the chrome://extensions/ page and click the Load unpacked extension button and select the unzipped folder for your extension to install it.
